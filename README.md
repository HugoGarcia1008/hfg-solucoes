# HFG Soluções — Landing Page

Landing page institucional da **HFG Soluções**, empresa de tecnologia, inovação e resultados especializada em software, sistemas, sites e soluções de TI para pequenos negócios.

## Stack

- HTML5
- CSS3 (tema futurista, glassmorphism, animações)
- JavaScript puro (sem frameworks/dependências)

## Estrutura

```
hfg-solucoes/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── assets/
    └── logo.jpeg
```

## Rodando localmente

Basta abrir `index.html` no navegador, ou usar um servidor estático, por exemplo:

```bash
npx http-server . -p 3001
```

## Deploy

Publicado via GitHub Pages a partir da branch `main`.
